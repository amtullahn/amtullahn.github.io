# Amtullah's Virtual Living Room Portfolio

A React + TypeScript Vite portfolio starter inspired by a whimsical living-room concept.

## Sections
- Welcome / gallery wall navigation
- About me
- Planned Substack posts (separate `/blog` page)
- Technical projects gallery with GitHub + demo links
- Heyzine flipbook embed area

## Run locally
```bash
npm install
npm run dev
```
